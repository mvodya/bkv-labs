Двоичные сортированные деревья

Задача - составить частотный словарь слов, используя набор файлов, 
используя двоичное дерево для создания и одновременной сортировки словаря.

Программа должна считывать несколько текстовых файлов, выделяя из входного потока
слова на русском языке и добавляя эти слова в двоичное дерево.
Подсчитывать количество появлений в тексте нужно увеличением переменной-члена
структуру УЗЕЛ (node). 
Словоформы не анализировать, как и в работе 2.

Вход программы - список файлов, которые необходимо обработать.
Выход программы - ФАЙЛ, в котором записанс частотный словарь, упорядоченный в
алфавитном порядке,

Использовать структуру вида

typedef struct node {
  int count;
  char *w;
  struct node *left;
  struct node *right;
} NODE;

NODE *tree=NULL ;

Реализовать функции поиска слова в дереве по первым 5 символам (использовать
strncmp(..) для английских слов  или ее аналог (написать самим!) для одной из  
русских кодировок , лучше для cp1251 для сравнения слов по первым символам).

ПОСТРОИТЬ ГРАФИКИ ЗАВИСИМОСТИ времени выполнения (составление словаря+вывод в файл)
от количества обработанных слов (в качестве входа брать разные наборы файлов)
для результатов второй лабработы и четвертой. Использовать электронные таблицы,
данные в них можно импортировать через копирование буфера обмена.
