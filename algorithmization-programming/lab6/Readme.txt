Алгоритм построения выпуклой оболочки (на плоскости)

Выбрать способ представления данных о точках и выпуклой оболочке в памяти.
Проще всего - задавать массивом структур
Прочитать из файла данные о наборе точек .

В каждой строке файла— три числа: номер точки и ее  координаты,  например:

1  1.2  3.1
2  0.5 2.2
3  0 0
4  1.5 1.5 
5  2.0 1.8
6  2.2 1.8

Конец данных определить по считыванию EOF.
